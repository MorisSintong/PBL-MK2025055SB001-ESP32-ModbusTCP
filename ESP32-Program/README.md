# Modbus TCP/IP Bridge Program for ESP32-S3
Dengan menggunakan module Ethernet W5500 yang kemudian dihubungkan dengan ESP32-S3 melalui komunikasi SPI, Maka ESP32 dapat berinteraksi dengan jaringan melalui interface RJ45 dan Kabel Ethernet.
Dengan ini memberikan kapabilitas untuk ESP32 agar dapat berkomunikasi dengan device didalam jaringan menggunakan protokol Modbus TCP/IP.
