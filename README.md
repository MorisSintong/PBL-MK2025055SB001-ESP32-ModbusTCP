# PBL-MK2025055SB001 - Assembly Station

Design and develop a SCADA system prototype using
ESP32 and Mitsubishi FX5U PLC as the main
components. This prototype is applied to the
Assembly Station with the main objective of this
project is to develop an automatic control and
monitoring system that can be accessed through Wi-
Fi By external devices, and through Modbus TCP/IP
communication, as well as supporting interface
integration with various devices such as PCs, ESP32,
and I/O modules.
